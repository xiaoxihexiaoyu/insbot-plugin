# 关键词搜索模式

## 使用方式

```
/insbot:keyword <账号> <搜索内容> [评论内容]
```

## 示例

```
/insbot:keyword 账号1 搜索 #ootd
/insbot:keyword 账号5 搜索 @competitor1
/insbot:keyword 账号2 搜索 fitness
/insbot:keyword 账号1 搜索 #fashion 评论"Nice!"
/insbot:keyword 账号1 搜索 #fitness AI评论
```
