# InstaBot 配置命令

## 查看配置

```
/insbot:config 查看配置
```

## 设置 AI API Key

```
/insbot:config 设置 AI key 为 sk-xxxxxxxxxxxx
```

## 设置账号序列号

```
/insbot:config 设置账号5序列号为 SERIAL_12345
```

## 配置文件位置

```
E:/work/echowwork/automat_review/insbot-config.json
```

## 获取 Kimi API Key

1. 访问 https://platform.moonshot.cn/console/api-keys
2. 登录或注册
3. 创建新的 API Key
4. 复制 Key 并用命令配置

## 示例

```
/insbot:config 设置 AI key 为 sk-abc123def456
/insbot:config 设置账号1序列号为 BROWSER_SERIAL_001
```
