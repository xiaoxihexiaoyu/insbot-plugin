# InstaBot 主命令

## 使用方式

```
/insbot:insbot <你的自然语言描述>
```

## 示例

```
/insbot:insbot 查看机器人状态
/insbot:insbot 用账号5搜索 #ootd，评论"Nice!"
/insbot:insbot 用 urls.csv 文件批量发送评论
/insbot:insbot 开启养号模式，浏览 #fashion 话题
/insbot:insbot 停止当前任务
```
